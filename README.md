### Installation
```bash
$ [sudo] pip install github-description
```

```bash
$ [sudo] npm i -g github-description
```

#### Config
```bash
$ export GITHUB_TOKEN="<GITHUB_TOKEN>"
```

#### Examples
```bash
$ python -m github_description repo "new desc"
$ python -m github_description repo
new desc
```
